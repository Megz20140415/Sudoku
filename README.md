###Sudoku game is a game developed by students in Computer Science - Helwan University.

##Game Features
* Solve and check Sudoku puzzles
<br />
<img align="center" width="60%" src="https://cloud.githubusercontent.com/assets/19598009/20634342/799797ae-b358-11e6-8dd0-174ad5334794.gif" >


* Save, load and delete Sudoku games
<br />
<img align="center" width="60%" src="https://cloud.githubusercontent.com/assets/19598009/20634199/cddbabf4-b356-11e6-8870-beb15eb23efa.gif">


* Track users solving time and make a score board
<br />
* Hints add +10 seconds to the timer
<br />
<img align="center" width="60%" src="https://cloud.githubusercontent.com/assets/19598009/20634340/73a17ffe-b358-11e6-9f70-b4048b2ecace.gif">


* Popups are used to notify the user with updates or errors
